Set name in telegram chat
=========================

# Launch
- prepare environment.
  
  ``python -m venv venv``

- activate environment

  ``./venv/bin/activate`` or ``./venv/Scripts/activate``
  
- install requirements.
  
  ``pip install -r requirements.txt``
  
- run.
  
  ``python -m set_name_in_telegram_chat.main``
